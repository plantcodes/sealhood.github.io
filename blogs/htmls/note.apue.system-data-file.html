Real-time extensions to POSIX.1 added support for multiple system clocks. A clock is identified by clockid_t type.

The clock_gettime function get the time of specified clock.

#include <sys/time.h>
int clock_gettime(clockid_t clock_id, struct timespec *tsp);

struct timespec expresses time values in terms of seconds and nanoseconds. Function int clock_getres(clockid_t clock_id, struct timespec *tsp) return resolution of a specified clock. Function int gettimeofday(struct timeval *restrict tp, void *restrict tzp) is now obsolescent(deprecated).
